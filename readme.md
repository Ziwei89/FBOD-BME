Flying Bird Object Detection in Surveillance Videos via Memory-Enhanced Aggregation  

Detecting flying birds in surveillance videos constitutes a challenging multimedia perception task characterized by low signal-to-noise ratios (SNR) and high morphological variability. To address the challenge of sustaining observations for such transient targets, this paper proposes FBOD-BME (Flying Bird Object Detection via Memory-Enhanced Spatio-Temporal Analysis). First, a Co-Attention Feature Aggregation module with Memory Enhancement (Co-Attention-FA-ME) is introduced to act as a temporal integrator, achieving sustained aggregation of spatiotemporal features at the model’s input. Subsequently, a dedicated detection network extracts targets from these enriched representations. During training, considering that flying birds exhibit significant variations in posture (e.g., walking, gliding, flapping), we incorporate a static center Gaussian distribution to constrain the dynamic SimOTA-OC label assignment. This strategy effectively models the spatial distribution characteristics of the avian target within the bounding box, mitigating misplaced supervision caused by morphological diversity. Extensive experiments on both the self-built FBD-SV-TSS dataset and the public FBD-SV-2024 dataset demonstrate that FBOD-BME significantly enhances detection performance for flying birds in surveillance video streams.
